このアプリはショッピングアプリです。スタッフ側とユーザー側に分かれユーザー登録ができます。
それぞれでユーザー登録ができ、スタッフ側は商品の登録ができ、ユーザー登録は商品の購入ができるという動きです。あくまで練習であり実際の売買に使用されることは想定していません。

このアプリは全てPHP（とHTML)で作成されており、フレームワークは利用していません。内容は本の「気づけばプロ並みPHP」を参考にして作成しました。現状CSSを利用していないのでかなりシンプルな画面になっています。

ユーザー登録やスタッフ登録、商品の登録や購入を行うと自動的にMySQLに保存されます。

セッションハイジャックなどの対策は行っていますが、まだまだセキュリティ対策は甘いところが多くあります。またPHPでの本格的な作成は初めてなので、非効率的な記載も多くあるかと思います。
